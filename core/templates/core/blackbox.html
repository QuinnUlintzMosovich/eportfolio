In class, we looked at a black box that we poured a certin amount of water into. The black box. The box is a mystery that has a unpredictable water input and output. The amount between each pour changes between tests, and and it always seems to change depending on the amount of water . The goal of the investigation was to make a model that replicate's the black box with a cardboard box insted or the plastic normol box to test a idea for the functionality of the black box.

Claim:

I feel that there is really not a lot of evidence when it comes to the black box seeing as we cant see inside .

Evidence:

The method that we gathered evidence when working with the mystery black box was to pour a sertin amount of water into the black box, then document the output of water from the box when less water came when more water came out etc.

The black box water output varies greatly, with an apparent pattern of less than 1 output to input ratio, or a zero for the same ratio. This is followed by extremely high (towards 2) output to input ratios. This also seems to vary depending on the amount of water imputed, and even varies without a change in water input. For example, in trial 2 for the black box, there is an input of 900 mL, but no output. However, for the subsequent 2 tests, the water output is greater than the input.

The model that we built (made of a funnel leading to a container, with a siphon leading to another container with another siphon leading out of the box) has comparatively consistent results with the same testing regime, with some tests (like test 8) having identical inputs and outputs. For example, in tests 1 and 2, the water output remanined the same (despite different imputs).

Reasoning

The definition of a model is something that is a simplified representation of something else.

The evidence that we gathered shows a black box with high variability in output. However, the data we have has highly variable input values that might contribute to the apparent variability. This might contribute to differences between the model box and the black box. However, some variability might be unable to be explained by this, such as the input being the same as the output for some tests. However, the testing for the model was also different from the testing for the black box, which might have affected the data. Overall, too many factors could have caused the difference in data between the model box and black box.

To prove that the model that we created is correct, I would need to know that the mystery black box could output the same as the input. This would get rid of one of the most major apparent differences between the model and black boxes.
